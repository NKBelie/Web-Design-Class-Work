Different types of selectors

There are 4 main types of selectors we can use in CSS

Element Selector :

This is when the tag was used to select an element in HTML document.

ID selector:

This is when one element was given an identifier which will differentiate it from others simular elements.

ClASS selector:

This is when a group of different elements are given an identifier to represent them from others.

PSEUDO selector:

This selector is used to represent a user activated state. it is mostly used for links.

Pseude Selector has the following components:

  1. a:link
  2. a:active
  3. a:visited
